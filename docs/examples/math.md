## Block Math:


This is a Block Formula: $$\sum_{i=0}^n i^2 = \frac{(n^2+n)(2n+1)}{6}$$

## Inline Math:

This is an Inline Formula: $\sum_{i=0}^n i^2 = \frac{(n^2+n)(2n+1)}{6}$.
